Update

Open Lab has a version history system. This lets you keep track of old version of for reference, so that you can always peek back at how you used to have things in a previous version.

  • Updates to your project are done in "revisions". A revision can add files, replace files, or delete files.
  • To add or replace a file, simply drag any number of files into the browser window, and drop them on the box that will appear. Alternatively, click the button that says "Upload Files".

Tip: The column on the left represents the existing files, the column on the right represents the updates you are making with this revision. If this is the first time you are adding files, there will simply be a help message replacing the column on the left.
